# Conway's Game of Life in C++

# Dependencies
[raylib](https://raylib.com)
on Arch Linux
`sudo pacman -S raylib`

# Quickstart

```
$ make
$ ./lifec
```

# Controls
Space to pause and unpause (when paused, left mouse button is draw, otherwise, it is to move around the grid)
Escape to quit  
R to randomise grid
C to clear grid
A to reset zoom
Scroll to zoom
