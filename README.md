# Conway's Game of Life in C++

# Details
I originally intended to be written in C but I decided to switch midway. The library used is [raylib](https://raylib.com).

# Dependencies
[raylib](https://raylib.com)
OpenGL

# Quickstart

```
$ make
$ ./lifec
```

# Controls
Space to pause  
Escape to quit  
R to reset grid
